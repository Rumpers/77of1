| T-02-02-06 | Info Disclosure | constitution.md content leaks via system prompt to fan | accept | Constitution IS deliberately injected into the LLM context (PERSONA-02 design); creator-side responsibility to keep secrets out of the file; L3 moderation in 02-05 catches accidental disclosure; future hardening = redact pass before prepend |
